# 29장 Math

```javascript
// 프로퍼티
.PI 원주율 반환

// 메서드, 암묵적 타입 변환이 이루어짐
.abs 숫자의 절대값 반환. 절대값은 반드시 0 또는 양수이어야 함
.round 숫자의 소수점 이하를 반올림한 정수로 반환
.ceil 올림한 정수로 반환
.floor 내림한 정수로 반환
.sqrt 제곱근 반환
.random 임의의 난수(0에서 1 미만의 실수) 반환
.pow(밑, 지수) 지수를 거듭제곱한 결과 반환
-> ES7에 도입된 지수 연산자 사용 2 ** 2 ** 2 //16
.max 인수 중 가장 큰 수를 반환, 배열을 인수로 받을 때는 .apply 메서드 또는 스프레드 문법을 사용
.min 인수 중 가장 작은 수 반환, 인수가 전달되지 않으면 Infinity 반환

```
